Como Rodar no Ubuntu

Salve os scripts (setup.sh e correcao.sh) na sua máquina virtual.

Dê permissão de execução a eles: chmod +x setup.sh correcao.sh

Para Iniciar a Prova: Roda o setup.sh (./setup.sh)

Para Corrigir: Roda o correcao.sh (./correcao.sh)

Para Limpar: Use o comando de limpeza simples: rm -rf /home/aluno/TRABALHO_BASH
